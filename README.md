# Number Recognition
Neural network dengan Python untuk multiclass image classification MNIST handwritten data by Yann Lecun

# Flask App

### Flask Pypi
`pip install Flask`

### Run app
`flask run` atau python `wsgi.py`


# Libraries
* Tensorflow 2.x
* Numpy
* Matplotlib
